# Woven Web Style Guide

## 1. Core Message

Woven Web is a non-profit organization dedicated to healing societal disconnection by harmoniously integrating technology, nature, and human connection. We serve community builders, technologists, and individuals seeking authentic connection in the Boulder area and beyond. Our unique approach combines ancestral wisdom with conscious technology use to create transformative experiences that bridge digital and physical worlds.

## 3. Design Direction

### Overall Aesthetic
Modern organic: Blending natural forms with clean technological elements
- Rounded corners
- Organic shapes with geometric precision
- Balanced asymmetry
- Nature-inspired textures

### Typography
- **Headers**: Neue Montreal (sans-serif)
  - Bold, grounded, modern
  - Used for primary headlines and navigation
- **Body**: Spectral (serif)
  - Organic, readable, warm
  - Used for longer text passages

### Visual Hierarchy
1. Emphasize connection points and intersections
2. Use organic spacing that feels natural rather than rigid
3. Layer information like mycological networks
4. Incorporate breathing room between elements

### Spacing Philosophy
- Asymmetrical but balanced
- Dynamic rather than static
- Inspired by natural growth patterns
- Minimum 24px spacing between major elements

### UI Component Personality
- Soft, organic transitions
- Natural state animations
- Tactile feedback
- Biomimetic loading states

## 4. Color Palette

### Primary Colors
- Earth Brown: `#8B5E3C`
  - Used for grounding elements and primary navigation
- Forest Green: `#2C5530`
  - Used for headers and key interactive elements

### Secondary Colors
- Mushroom Gray: `#9A8F85`
  - Used for supporting elements and backgrounds
- Sky Blue: `#7CA5B8`
  - Used for highlights and secondary interactions

### Accent Colors
- Moon Gold: `#C9B037`
  - Used sparingly for emphasis and special elements
- Sunset Orange: `#E85D3F`
  - Used for calls-to-action and important notifications

### Neutral Tones
- Stone Light: `#F5F2ED`
  - Primary background color
- Stone Dark: `#2F2F2F`
  - Primary text color

### Usage Guidelines
1. Maintain 60-30-10 ratio (primary-secondary-accent)
2. Ensure WCAG 2.1 AA compliance for all text combinations
3. Use colors in organic gradients rather than solid blocks
4. Layer colors with subtle transparency for depth
5. Incorporate natural color transitions in interactive states
